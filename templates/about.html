Time History screening 1.0

This is screening of transient timepoints from the given inputs .

Home

This web app will screen the data based on three criteria. The criteria used are peak point selection, valley point selection and value of delta difference between two points.

Input Requirements: The input excel sheet should contain raw time points data with sheet named "Time vs Temp". the input excel sheet should contain delta difference between two points data with sheet named "Temperature difference" .

The Input excel file should contain sheet: "Temp vs Time", and row 4 should start with "Time,sec" as shown in picture.



The Input excel file should contain sheet: "Temperature difference" as shown in picture.

Select the input excel file using the Choose file button in the home page and submit the execute button, the output file will be saved as filename.xlsx.
